# Day3: 基本データ型（Int, Double, String, Boolean）

## 問題

学生の成績情報を管理するプログラムを作成してください。

## 要件

1. 以下の変数を適切な型で宣言してください：
   - `studentName`: 学生の名前（文字列）→ "田中太郎"
   - `studentId`: 学生番号（整数）→ 12345
   - `mathScore`: 数学の点数（小数）→ 85.5
   - `englishScore`: 英語の点数（小数）→ 92.0
   - `isPassed`: 合格判定（真偽値）→ 平均点が60点以上なら合格

2. 平均点を計算し、`average`変数に格納してください

3. 以下の形式で出力してください：
   - 学生名: [名前]
   - 学生番号: [番号]
   - 数学: [点数]点
   - 英語: [点数]点
   - 平均: [平均点]点
   - 合格判定: [true/false]

## 期待される出力

```
学生名: 田中太郎
学生番号: 12345
数学: 85.5点
英語: 92.0点
平均: 88.75点
合格判定: true
```

## ヒント

- 平均点は `(mathScore + englishScore) / 2` で計算できます
- 比較演算子 `>=` を使って合格判定を行います
- 文字列テンプレート `$変数名` や `${式}` を使うと便利です
