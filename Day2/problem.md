# Day2: 変数宣言（val/var）

## 問題

変数宣言を使って、自己紹介プログラムを作成してください。

## 要件

1. `val`を使って以下の変数を宣言する：
   - `name`: 自分の名前（String型）
   - `birthYear`: 生まれた年（Int型）

2. `var`を使って以下の変数を宣言する：
   - `hobby`: 趣味（String型、最初は「プログラミング」）

3. `hobby`の値を「Kotlin学習」に変更する

4. 現在の年（2024）を使って年齢を計算し、`age`変数に格納する（`val`を使用）

5. 以下の形式で出力する：
   ```
   名前: [name]
   年齢: [age]歳
   趣味: [hobby]
   ```

## ヒント

- 年齢の計算: `現在の年 - 生まれた年`
- 文字列の中に変数を埋め込むには `$変数名` を使う（文字列テンプレート）

## 期待される出力例

```
名前: 太郎
年齢: 24歳
趣味: Kotlin学習
```

※名前と年齢は自分の情報に置き換えてください
